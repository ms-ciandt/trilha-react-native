# iOS Developer Trail — Contexto Local

## Público-alvo

Devs iOS com experiência em Swift e SwiftUI, sem experiência em JavaScript/React.

## Princípio central

Todo tópico parte do que o dev já sabe em Swift/SwiftUI e mapeia para o equivalente React Native.
As analogias são sempre Swift/SwiftUI → RN, nunca Kotlin → RN (para isso existe trilha-nativo).
