## Module Map

### 1. Brownfield Integration

Adding React Native to an existing native app. Host apps, bridge setup, routing between native and RN screens.

---

### 2. TurboModules

Writing high-performance native modules with the New Architecture. JSI bindings, code generation, typed interfaces.

---

### 3. Fabric & JSI in Deep

Internals of the new rendering pipeline. C++ layer, synchronous calls, shadow tree, concurrent features.

---

### 4. Performance, Bundle & CI/CD

Metro bundler configuration, Hermes, profiling, startup time, release pipelines and automated delivery.

---

### 5. Maintenance: React Native Version Updates

Strategies for upgrading RN versions safely. Dependency management, breaking changes, tooling.
