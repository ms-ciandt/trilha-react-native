# Trilha Nativo

Para desenvolvedores Android (Kotlin) e iOS (Swift) que querem aprender React Native.

Você já entende rendering nativo, threads, lifecycle e APIs de plataforma — aqui você aprende JavaScript/TypeScript, React e como seu conhecimento nativo se mapeia para o mundo RN.
